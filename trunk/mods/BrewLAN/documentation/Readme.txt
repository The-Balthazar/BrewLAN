--------------------------------------------------------------------------------
-- Intro
--------------------------------------------------------------------------------
BrewLAN is a mod for Supreme Commander: Forged Alliance that has been in on and
off development by a team of one for over half a decade. It adds a wide variety
of new units for all factions, as well as offering several new game modes and
features. The core design goals for the mod are simple; to add new, interesting,
and visually aesthetic units which complement those already in existence and to
maximize technical compatibility with other mods.

--------------------------------------------------------------------------------
-- Installation
--------------------------------------------------------------------------------
If you have found this file, then you have probably already installed. If not
then remove any old version of BrewLAN and its sub-mods, if applicable, then
merge the contents of the \mods\ and \gamedata\ folders with that of your games'
root directory. The mods folder may not already exist if this is your first time
installing a mod. If done correctly the following should exist:

  "\supreme commander forged alliance\mods\BrewLAN\mod_info.lua"
  "\supreme commander forged alliance\gamedata\brewlan.scd"

For additional help see: http://code.google.com/p/brewlan/wiki/Downloads?tm=2
